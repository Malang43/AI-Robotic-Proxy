# Xavier NX Setup Steps

## Status
Pending hardware availability.

## Planned Steps
1. Connect Xavier NX to power, monitor, keyboard, and network.
2. Boot the device or enter recovery mode if flashing is required.
3. Use NVIDIA SDK Manager for JetPack installation if needed.
4. Complete Ubuntu first boot setup.
5. Install required ROS2 version according to JetPack/Ubuntu compatibility.
6. Verify terminal access and system information.
7. Connect robot peripherals such as LIDAR and motor controller.
8. Document all configuration steps with screenshots.
